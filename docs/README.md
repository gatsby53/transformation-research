# Transformation Research

## Purpose

本リポジトリは、変身企業（Transformation Company）の定量研究を行うための研究基盤である。

目的は以下。

* 超成功企業の共通点を発見する
* 第一成功、第一変身、第二変身の特徴を定量化する
* 多段変身企業の成長パターンを理解する
* 投資理論「Transformation Theory」を構築する

---

## Research Questions

* 超成功企業はどのような条件で生まれるのか？
* 本業が健全なうちに第二の柱を作る企業は強いのか？
* 多段変身企業の共通点は何か？
* 第一成功時点で将来の超成功企業を見抜けるのか？
